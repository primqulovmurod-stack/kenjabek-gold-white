# Top 10 DevOps Repositories

1. **bregman-arie/devops-resources**
   - [Link](https://github.com/bregman-arie/devops-resources)
   - Large collection of DevOps resources.
2. **MichaelCade/90DaysOfDevOps**
   - [Link](https://github.com/MichaelCade/90DaysOfDevOps)
   - A guide to learning the foundations of DevOps in 90 days.
3. **docker/docker-ce**
   - [Link](https://github.com/docker/docker-ce)
   - Docker Community Edition.
4. **kubernetes/kubernetes**
   - [Link](https://github.com/kubernetes/kubernetes)
   - Production-Grade Container Scheduling and Management.
5. **hashicorp/terraform**
   - [Link](https://github.com/hashicorp/terraform)
   - Infrastructure as Code tool.
6. **ansible/ansible**
   - [Link](https://github.com/ansible/ansible)
   - Simple IT automation and configuration management.
7. **prometheus/prometheus**
   - [Link](https://github.com/prometheus/prometheus)
   - The Prometheus monitoring system and time series database.
8. **trimstray/the-book-of-secret-knowledge**
   - [Link](https://github.com/trimstray/the-book-of-secret-knowledge)
   - A collection of inspiring lists, manuals, cheatsheets, blogs, hacks, one-liners, cli/web tools.
9. **bregman-arie/devops-exercises**
   - [Link](https://github.com/bregman-arie/devops-exercises)
   - Linux, Jenkins, Docker, K8s... 2600+ Exercises and Questions.
10. **grafana/grafana**
    - [Link](https://github.com/grafana/grafana)
    - The open-source platform for monitoring and observability.
