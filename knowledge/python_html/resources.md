# Top 10 Python & HTML Repositories

## Python
1. **tensorflow/tensorflow**
   - [Link](https://github.com/tensorflow/tensorflow)
   - Machine Learning for everyone.
2. **pytorch/pytorch**
   - [Link](https://github.com/pytorch/pytorch)
   - Tensors and Dynamic neural networks in Python with strong GPU acceleration.
3. **keras-team/keras**
   - [Link](https://github.com/keras-team/keras)
   - Deep Learning for humans.
4. **scikit-learn/scikit-learn**
   - [Link](https://github.com/scikit-learn/scikit-learn)
   - Machine learning in Python.
5. **psf/requests**
   - [Link](https://github.com/psf/requests)
   - HTTP for Humans.

8. **jgthms/bulma**

9. **twbs/bootstrap**
   - [Link](https://github.com/twbs/bootstrap)

10. **andy-piccalilli/modern-css-reset**
    - [Link](https://github.com/andy-piccalilli/modern-css-reset)
