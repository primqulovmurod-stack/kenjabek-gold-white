# Top 10 Web Designer Repositories

1. **Designing-Apps/design-resources**
   - [Link](https://github.com/Designing-Apps/design-resources)
   - Curated list of design resources for apps.
2. **google/material-design-icons**
   - [Link](https://github.com/google/material-design-icons)
   - Official open source icons from Google.
3. **bradfrost/style-guide-guide**
   - [Link](https://github.com/bradfrost/style-guide-guide)
   - A guide to creating style guides and design systems.
4. **juliangarnier/anime**
   - [Link](https://github.com/juliangarnier/anime)
   - JavaScript animation engine.
5. **florinpop17/design-resources-for-developers**
   - [Link](https://github.com/florinpop17/design-resources-for-developers)
   - A list of design resources for developers.
6. **tabler/tabler-icons**
   - [Link](https://github.com/tabler/tabler-icons)
   - A set of over 5000 free-to-use SVG icons.
7. **Lucide-icons/lucide**
   - [Link](https://github.com/lucide-icons/lucide)
   - Clean, lightweight, and customizable open source icons.
8. **primer/design**
   - [Link](https://github.com/primer/design)
   - GitHub's design system resources.
9. **feathericons/feather**
   - [Link](https://github.com/feathericons/feather)
   - Simply beautiful open source icons.
10. **design-system-internet/design-system**
    - [Link](https://github.com/design-system-internet/design-system)
    - Modern web design standards and resources.
