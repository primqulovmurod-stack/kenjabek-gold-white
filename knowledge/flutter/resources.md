# Top 10 Flutter Repositories

1. **flutter/flutter**
   - [Link](https://github.com/flutter/flutter)
   - Flutter makes it easy and fast to build beautiful apps for mobile and beyond.
2. **flutter/samples**
   - [Link](https://github.com/flutter/samples)
   - A collection of Flutter samples and demos.
3. **Solido/awesome-flutter**
   - [Link](https://github.com/Solido/awesome-flutter)
   - An awesome list that curates the best Flutter libraries, tools, tutorials, articles and more.
4. **felangel/bloc**
   - [Link](https://github.com/felangel/bloc)
   - A predictable state management library that helps implement the BLoC design pattern.
5. **rrousselGit/riverpod**
   - [Link](https://github.com/rrousselGit/riverpod)
   - A reactive caching and data-binding framework.
6. **AppFlowy-IO/AppFlowy**
   - [Link](https://github.com/AppFlowy-IO/AppFlowy)
   - AppFlowy is an open-source alternative to Notion.
7. **localsend/localsend**
   - [Link](https://github.com/localsend/localsend)
   - An open-source cross-platform alternative to AirDrop.
8. **GetRexZ/getx**
   - [Link](https://github.com/GetRexZ/getx)
   - High-performance state management, intelligent dependency injection, and route management.
9. **iampawan/FlutterExampleApps**
   - [Link](https://github.com/iampawan/FlutterExampleApps)
   - [Example] Basic Flutter apps for beginners.
10. **Tencent/tdesign-flutter**
    - [Link](https://github.com/Tencent/tdesign-flutter)
    - A comprehensive Flutter UI component library.
