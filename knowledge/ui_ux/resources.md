# UI/UX & Animation Skills for Taklifnoma Asia

This document contains the 10 best UI/UX and animation libraries/components to make "Taklifnoma Asia" a premium experience.

## 1. Aceternity UI
- **URL:** [https://github.com/aceternity/ui](https://github.com/aceternity/ui)
- **Description:** High-end, "wow-factor" components like 3D cards, spotlight effects, and animated backgrounds.
- **Top Components for Invitations:**
    - **Bento Grid:** For structured info layouts.
    - **3D Card Effect:** For interactive invitation cards.
    - **Background Beams:** For a luxury background feel.

## 2. Magic UI
- **URL:** [https://github.com/magicuidesign/magicui](https://github.com/magicuidesign/magicui)
- **Description:** Animated marketing components built with Framer Motion and Tailwind CSS.
- **Use Case:** Creating high-conversion landing pages and interactive elements.

## 3. Framer Motion
- **URL:** [https://github.com/framer/motion](https://github.com/framer/motion)
- **Description:** The industry standard for React animations.
- **Key Features:** `AnimatePresence`, `useScroll`, and layout transitions.

## 4. GSAP (GreenSock)
- **URL:** [https://github.com/greensock/GSAP](https://github.com/greensock/GSAP)
- **Description:** Professional-grade animation engine for complex sequences.
- **Use Case:** Complex scroll-triggered animations.

## 5. Locomotive Scroll / Lenis
- **URL:** [https://github.com/darkroomengineering/lenis](https://github.com/darkroomengineering/lenis)
- **Description:** Smooth scrolling libraries that give a premium "luxury" feel.

## 6. Canvas Confetti
- **URL:** [https://github.com/catdad/canvas-confetti](https://github.com/catdad/canvas-confetti)
- **Description:** Lightweight confetti explosion effect.
- **Use Case:** Celebration moments when an invitation is accepted.

## 7. Swiper.js
- **URL:** [https://github.com/nolimits4web/swiper](https://github.com/nolimits4web/swiper)
- **Description:** The best modern slider for galleries.
- **Use Case:** Photo galleries of the event/couple.

## 8. React Parallax Tilt
- **URL:** [https://github.com/mkosir/react-parallax-tilt](https://github.com/mkosir/react-parallax-tilt)
- **Description:** Interactive tilt effect for cards and images.

## 9. AOS (Animate On Scroll)
- **URL:** [https://github.com/michalsnik/aos](https://github.com/michalsnik/aos)
- **Description:** Simple library to animate elements as they enter the viewport.

## 10. Lucide Icons
- **URL:** [https://github.com/lucide-icons/lucide](https://github.com/lucide-icons/lucide)
- **Description:** Beautiful, consistent, and lightweight vector icons.
